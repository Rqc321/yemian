使用帮助

基本操作

  • 直接在编辑器中输入或粘贴文本
  • 所有更改会自动保存，无需手动保存
  • 点击"复制链接"按钮分享文档给他人
  • 使用顶部菜单可以创建新文档、下载文档等

功能特点

  • 无需注册登录，直接使用
  • 支持700万字的超大文本存储
  • 多人可同时编辑同一文档
  • 自动保存，防止意外关闭丢失内容
  • 可自定义字体、字号、行高等样式
  • 支持浅色/深色主题切换

快捷键

Ctrl + S 手动保存
Ctrl + N 新建文档
Ctrl + L 复制链接
F11 全屏编辑
Ctrl + / 显示/隐藏帮助

注意事项

  • 文档链接可被任何人访问和编辑，请谨慎分享
  • 建议定期导出重要文档备份
  • 超大文本可能会影响编辑性能
  • 清除浏览器数据可能会导致本地存储的文档丢失
